# Security Policy

## Supported Versions

Only the latest major release is supported.

| Version | Supported |
|---------|-----------|
| 3.x     | Yes       |
| 2.x     | No        |
| 1.x     | No        |

## Reporting a Vulnerability

If you need to report a vulnerability, you can send an e-mail to
[contact@rudloff.pro](mailto:contact@rudloff.pro).
